# Учебный проект №2 — "Магазин"

### Порядок установки проекта

	1. git clone %git_repository% %your_folder%
	2. cd %your_folder%
	3. bower i
	4. npm i
	5. gulp

### Features ###
Интерактив: ховер на корзину, слайдер картинок товара и кнопка скролла вверх
